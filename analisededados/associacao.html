Referência Rápida

Quando usar Qui-Quadrado?

  • ✓ Variáveis categóricas (nominais ou ordinais)
  • ✓ Frequências esperadas ≥ 5 em todas as células
  • ✓ Amostras grandes (n > 40)
  • ✓ Observações independentes

Quando usar Teste de Fisher?

  • ✓ Tabelas 2×2 (duas variáveis dicotômicas)
  • ✓ Amostras pequenas (n < 40)
  • ✓ Frequências esperadas < 5
  • ✓ Quando χ² não é apropriado

Interpretando os Resultados

P-valor

Probabilidade de observar os dados se H₀ for verdadeira

p < 0.05: Rejeitar H₀ (há associação)

p ≥ 0.05: Não rejeitar H₀

V de Cramér

Força da associação (0 a 1)

0.0 - 0.1: Muito fraca

0.1 - 0.3: Fraca

0.3 - 0.5: Moderada

0.5+: Forte

Resíduos

Identificam células que contribuem para χ²

|resíduo| > 2: Célula significativa
