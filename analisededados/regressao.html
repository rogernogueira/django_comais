Referência Rápida

Quando usar Regressão Linear?

  • ✓ Variável resposta é contínua (números)
  • ✓ Relação linear entre X e Y
  • ✓ Resíduos normalmente distribuídos

Quando usar Regressão Logística?

  • ✓ Variável resposta é categórica (Sim/Não)
  • ✓ Quer estimar probabilidades
  • ✓ Classificação binária ou multinomial
