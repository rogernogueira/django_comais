Anscombe's Quartet

A prova definitiva de que você DEVE plotar seus dados

O Paradoxo

Francis Anscombe (1973) criou quatro conjuntos de dados que têm estatísticas descritivas quase idênticas, mas aparências completamente diferentes.

Estatísticas Idênticas para os 4 grupos:

  • Média de X: 9.0
  • Média de Y: 7.50
  • Variância de X: 11.0
  • Correlação (r): 0.816
  • Reta de Regressão: y = 3 + 0.5x

A Lição

Se você olhasse apenas para os números ("r = 0.81"), diria que os quatro casos são iguais.

Mas ao olhar os gráficos, vemos que apenas um é uma relação linear "normal". Os outros têm problemas graves (não linearidade, outliers extremos) que invalidam a correlação de Pearson.
