Histograma e Densidade

Vendo a "alma" dos dados

Histograma

Agrupa os dados em "bins" (faixas) e conta a frequência.

  • Mostra a distribuição de frequência real.
  • Sensível à escolha do número de faixas (bins).
  • Ótimo para ver gaps nos dados.

Densidade (KDE)

Uma versão suavizada do histograma. Estima a probabilidade.

  • Melhor para visualizar a "forma" abstrata.
  • Permite comparar múltiplos grupos sobrepostos.
  • Esconde pequenos detalhes/ruídos.

Quando usar qual?

Use Histograma: Para exploração inicial e verificar a qualidade dos dados (ex: ver se números foram arredondados).
Use Densidade: Para apresentações finais ou comparar a distribuição de 2+ grupos no mesmo gráfico.
