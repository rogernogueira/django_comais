Normalidade dos Resíduos

Os erros do modelo devem seguir uma distribuição normal.

QQ-Plot (Quantil-Quantil)

Se os pontos seguem a linha diagonal, os resíduos são normais.

Histograma dos Resíduos

Deve ter formato de "sino" (simétrico).

Testes Estatísticos

Shapiro-Wilk

Recomendado para amostras pequenas (n < 50). É o teste mais poderoso para normalidade.

Kolmogorov-Smirnov

Recomendado para amostras grandes (n > 50). Geralmente usado com a correção de Lilliefors.

Anderson-Darling

Mais sensível nas caudas da distribuição que o KS.

Interpretação do P-valor

p < 0.05

NÃO é normal

(Rejeita H0)

p ≥ 0.05

É normal

(Não rejeita H0)

H0 (Hipótese Nula): Os resíduos seguem uma distribuição normal.

⚠️ Nota Importante

Não é necessário que X ou Y sejam normais, apenas os resíduos. Para amostras grandes (n > 200), a regressão é robusta a violações leves de normalidade (Teorema Central do Limite).

O que fazer se os resíduos não forem normais?

Transformação: Aplicar Log(Y) ou √Y

Aumentar n: Coletar mais dados
