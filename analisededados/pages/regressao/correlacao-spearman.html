Correlação de Spearman (ρ)

Medindo relações monotônicas usando rankings.

O que é Relação Monotônica?

Uma relação é monotônica quando, à medida que X aumenta, Y sempre aumenta (ou sempre diminui), mas não necessariamente de forma linear.

Linear

y = 2x + 1

✓ Monotônica

Exponencial

y = e^x

✓ Monotônica

Logarítmica

y = log(x)

✓ Monotônica

Quadrática

y = x²

✗ Não monotônica

Como Funciona?

Passo 1: Converta os valores em rankings (posições)

X: [10, 20, 30, 40] → Rank: [1, 2, 3, 4]

Y: [5, 15, 25, 35] → Rank: [1, 2, 3, 4]

Passo 2: Calcule Pearson nos rankings

ρ = correlação de Pearson entre os rankings

Fórmula de Spearman

ρ = 1 - (6Σdᵢ²) / (n(n² - 1))

dᵢ = diferença entre os rankings

n = número de pares

Esta é a fórmula simplificada. Quando há empates, usa-se a correlação de Pearson nos rankings.

Interpretação

ρ ≈ +1 Monotônica crescente perfeita
ρ ≈ 0 Sem relação monotônica
ρ ≈ -1 Monotônica decrescente perfeita

Vantagens

  • ✓ Detecta relações não-lineares monotônicas
  • ✓ Robusto a outliers
  • ✓ Funciona com dados ordinais
  • ✓ Não assume normalidade

Quando Usar Spearman?

✓ Use quando:

  • • A relação é monotônica mas não-linear
  • • Dados ordinais (rankings, escalas Likert)
  • • Presença de outliers
  • • Distribuição não-normal
  • • Dados com valores extremos

⚠️ Limitações:

  • • Não detecta relações não-monotônicas (ex: U invertido)
  • • Perde informação ao converter para rankings
  • • Menos poderoso que Pearson para dados lineares
