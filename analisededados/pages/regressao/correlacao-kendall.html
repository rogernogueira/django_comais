Correlação de Kendall (τ)

Medindo concordância entre rankings.

O que é Tau de Kendall?

O coeficiente Tau de Kendall (τ) mede a concordância entre dois rankings, contando pares concordantes e discordantes.

Pares Concordantes vs Discordantes

Concordante:

Se X₁ > X₂, então Y₁ > Y₂

Discordante:

Se X₁ > X₂, mas Y₁ < Y₂

Exemplo Visual

Dados: (1,1), (2,2), (3,4), (4,3)

(1,1) vs (2,2) ✓ Concordante
(1,1) vs (3,4) ✓ Concordante
(3,4) vs (4,3) ✗ Discordante

Total: 5 concordantes, 1 discordante

Fórmula de Kendall

τ = (C - D) / (C + D)

C = número de pares concordantes

D = número de pares discordantes

Nota: Existem variantes (τ-a, τ-b, τ-c) para lidar com empates. τ-b é a mais comum.

Interpretação

τ = +1 Concordância perfeita
τ = 0 Sem associação
τ = -1 Discordância perfeita

Características

  • ✓ Mais robusto que Spearman
  • ✓ Melhor para amostras pequenas
  • ✓ Interpretação direta (probabilidade de concordância)
  • ✓ Funciona bem com empates

Kendall vs Spearman

Kendall (τ)

  • ✓ Mais robusto estatisticamente
  • ✓ Melhor para amostras pequenas (n < 30)
  • ✓ Interpretação mais intuitiva
  • ✓ Valores geralmente menores que Spearman
  • ✗ Mais lento de calcular (O(n²))

Spearman (ρ)

  • ✓ Mais rápido de calcular
  • ✓ Melhor para amostras grandes
  • ✓ Valores mais próximos de Pearson
  • ✓ Mais conhecido e usado
  • ✗ Menos robusto que Kendall

Quando Usar Kendall?

✓ Use quando:

  • • Amostras pequenas (n < 30)
  • • Muitos empates nos dados
  • • Precisa de interpretação intuitiva
  • • Quer máxima robustez
  • • Dados ordinais

⚠️ Considere Spearman se:

  • • Amostras grandes (n > 100)
  • • Velocidade de cálculo é importante
  • • Quer valores comparáveis a Pearson
