Regressão Linear Simples

Modelando a relação entre uma variável dependente (Y) e uma preditora (X).

O que é?

A Regressão Linear Simples modela a relação entre duas variáveis usando uma linha reta. É o método mais básico e fundamental de regressão.

Y = β₀ + β₁X + ε

β₀ = intercepto (onde a linha cruza o eixo Y)

β₁ = inclinação (mudança em Y por unidade de X)

ε = erro (resíduo)

Interpretação

Intercepto (β₀)

Valor esperado de Y quando X = 0

Inclinação (β₁)

Para cada aumento de 1 unidade em X, Y aumenta β₁ unidades

Exemplo: Y = 30 + 2X (Salário vs Anos de Experiência)
Salário inicial = R$ 30k, cada ano adiciona R$ 2k

Método dos Mínimos Quadrados

A linha de regressão minimiza a soma dos quadrados dos resíduos (SSE).

SSE = Σ(yᵢ - ŷᵢ)²

Menor SSE = melhor ajuste

Quando Usar?

✓ Use quando:

  • • Você tem apenas uma variável preditora
  • • A relação parece linear no scatterplot
  • • Você quer entender a relação básica entre X e Y

✗ Não use quando:

  • • A relação é claramente curva (não-linear)
  • • Você tem múltiplas variáveis importantes
  • • Y é categórico (use Regressão Logística)
