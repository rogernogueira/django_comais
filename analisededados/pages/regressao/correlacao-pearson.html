Fórmula de Pearson

r = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / √[Σ(xᵢ - x̄)² · Σ(yᵢ - ȳ)²]

Covariância dividida pelo produto dos desvios padrão

O que é?

A correlação de Pearson (r) mede a força e direção da relação linear entre duas variáveis contínuas. O coeficiente varia de -1 a +1:

r = +1 Correlação positiva perfeita
r = 0 Sem correlação linear
r = -1 Correlação negativa perfeita

Interpretação da Força

|r| > 0.7 Forte
0.3 < |r| < 0.7 Moderada
|r| < 0.3 Fraca

Características

  • ✓ Mede apenas relações lineares
  • ✓ Sensível a outliers
  • ✓ Requer dados contínuos
  • ✓ Assume distribuição aproximadamente normal

Quando Usar Pearson?

✓ Use quando:

  • • Ambas variáveis são contínuas
  • • A relação é linear
  • • Dados aproximadamente normais
  • • Sem outliers extremos

✗ Não use quando:

  • • A relação é não-linear (curva)
  • • Dados ordinais (rankings)
  • • Muitos outliers presentes
  • • Distribuição muito assimétrica

⚠️ Correlação ≠ Causalidade

Uma correlação forte não significa que uma variável causa a outra!

Exemplo: Vendas de sorvete e afogamentos têm correlação positiva.

Mas o sorvete não causa afogamentos! Ambos aumentam no verão (variável confundidora).

Lembre-se: Correlação é apenas o primeiro passo.

Para estabelecer causalidade, você precisa de experimentos controlados ou análises mais avançadas.
