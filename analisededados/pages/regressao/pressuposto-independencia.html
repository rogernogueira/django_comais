Independência dos Erros

Os erros não devem estar correlacionados entre si.

Teste de Durbin-Watson

d = Σ(eᵢ - eᵢ₋₁)² / Σeᵢ²

d = estatística Durbin-Watson

e = resíduos

0
2 (Ideal)
4

O que é Autocorrelação?

Quando o erro de hoje está correlacionado com o erro de ontem. Comum em dados de séries temporais (vendas mensais, temperatura diária, etc).

⚠️ Se houver autocorrelação, seus intervalos de confiança serão muito estreitos (otimistas demais).

Interpretação do Durbin-Watson

d ≈ 2

Sem autocorrelação. Tudo OK!

d < 1.5

Autocorrelação Positiva. Erros consecutivos tendem a ter o mesmo sinal.

d > 2.5

Autocorrelação Negativa. Erros consecutivos tendem a alternar de sinal.

Soluções

Modelos de Séries Temporais

Use ARIMA ou modelos autorregressivos que modelam explicitamente a dependência temporal.

Adicionar Variáveis de Lag

Inclua valores defasados (t-1, t-2) como preditores.

Verificar Variáveis Omitidas

Pode haver uma variável importante faltando que explica o padrão temporal.

Quando este pressuposto é mais crítico?

Séries Temporais: Dados coletados ao longo do tempo

Dados Espaciais: Observações geograficamente próximas

Medidas Repetidas: Múltiplas observações do mesmo indivíduo
