Quando Usar o Teste Exato de Fisher?

O padrão-ouro para análise de associação em amostras pequenas

Cenário Principal: Amostras Pequenas

O Teste Exato de Fisher é fundamental quando o tamanho da amostra é insuficiente para garantir a precisão da aproximação assintótica do Qui-Quadrado.

Regra de Ouro:

Use Fisher sempre que qualquer célula da tabela de contingência tiver uma frequência esperada menor que 5.

Checklist de Aplicação

  • ✓
    Tabelas 2x2:

    Embora existam extensões, é classicamente usado para duas variáveis dicotômicas.

  • ✓
    Dados Esparsos:

    Quando você tem muitos zeros ou números muito baixos (1, 2) na tabela.

  • ✓
    Desequilíbrio:

    Quando uma categoria é muito rara em comparação com a outra (ex: 95% vs 5%).

Exemplo: Estudo Clínico Raro

Testando um novo medicamento em uma doença ultra-rara.

Grupo Curado Não Curado
Tratamento 4 1
Placebo 1 5

⚠️ Frequências muito baixas! O Qui-Quadrado falharia aqui. Fisher é obrigatório.

Por que não usar sempre?

Se é "exato", por que não usar para tudo?

  • Custo computacional alto para amostras grandes.
  • Para N grande, o Qui-Quadrado converge para o mesmo resultado.
  • Pode ser conservador demais em alguns desenhos de estudo.
