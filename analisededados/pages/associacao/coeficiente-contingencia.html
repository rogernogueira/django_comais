Coeficiente de Contingência (C)

Uma medida clássica baseada no Qui-Quadrado

O que é?

O Coeficiente de Contingência de Pearson (C) é uma medida de associação para variáveis nominais. Ele tenta ajustar o valor do Qui-Quadrado para o tamanho da amostra.

Limitação Importante:

O valor máximo de C nunca atinge 1, mesmo com associação perfeita. O valor máximo possível depende do número de linhas e colunas da tabela.

Fórmula

$$ C = \sqrt{\frac{\chi^2}{\chi^2 + n}} $$

Onde χ² é a estatística Qui-Quadrado e n é o tamanho total da amostra.

Quando Usar?

  • ✓ Para tabelas maiores que 2x2 onde se deseja uma medida simples.
  • ✓ Apenas para comparar tabelas de mesma dimensão (ex: comparar duas tabelas 3x3).

Devido à sua limitação de não atingir 1, o V de Cramér é geralmente preferido na estatística moderna.

Exemplo de Limitação

Em uma tabela 2x2, o valor máximo possível para C é aproximadamente 0.71.

Isso significa que um C = 0.71 em uma tabela 2x2 indica uma associação perfeita, o que pode ser confuso se interpretado como uma correlação normal (onde 0.71 seria apenas "forte").
