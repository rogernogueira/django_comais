Tamanho Amostral

O impacto do N na escolha do teste e na confiabilidade

Regra Geral

Para o teste Qui-Quadrado, não existe um número mágico fixo para o tamanho total da amostra (N), mas ele deve ser grande o suficiente para satisfazer o pressuposto das frequências esperadas.

Diretrizes Comuns:

  • N > 40: Qui-Quadrado é quase sempre seguro.
  • 20 < N < 40: Qui-Quadrado é aceitável se todas as frequências esperadas forem ≥ 5.
  • N < 20: Use o Teste Exato de Fisher.

O Paradoxo do N Grande

O teste Qui-Quadrado é muito sensível ao tamanho da amostra.

Cuidado:

Com amostras muito grandes (ex: N > 1000), até mesmo associações minúsculas e irrelevantes se tornarão estatisticamente significativas (p < 0.05).

Por isso, é crucial calcular o Tamanho do Efeito (V de Cramér ou Phi) para saber se a associação é relevante na prática.

Poder Estatístico

Amostras muito pequenas podem não ter poder suficiente para detectar uma associação real (Erro Tipo II - Falso Negativo).

Se você tem uma tabela com muitas células (ex: 4x5) e um N moderado, as observações ficarão muito "espalhadas", resultando em frequências esperadas baixas e perda de poder.

Recomendação Prática

Ao planejar um estudo:

  • ✓ Tente ter pelo menos 5 a 10 observações por célula da tabela.
  • ✓ Ex: Para uma tabela 3x3 (9 células), tente um N total de pelo menos 45 a 90.
