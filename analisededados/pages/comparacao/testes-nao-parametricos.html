Comparação de Distribuições

Quando Usar Testes Não Paramétricos?

⚠️ Dados não normais

⚠️ Presença de outliers

⚠️ Amostras pequenas (n < 30)

⚠️ Dados ordinais/categóricos

O que são Testes Não Paramétricos?

Testes não paramétricos são métodos estatísticos que não assumem distribuição normal dos dados. Eles são robustos a outliers e funcionam bem com amostras pequenas e dados ordinais.

Teste Mann-Whitney U

✓ Compara 2 grupos independentes

✓ Usa postos (ranks) em vez de valores

✓ Alternativa ao teste t independente

✓ Testa diferenças nas medianas

Teste Wilcoxon

✓ Compara 2 grupos pareados

✓ Mesmo grupo em momentos diferentes

✓ Alternativa ao teste t pareado

✓ Testa diferenças nas medianas

Vantagens dos Testes NP

Robustez

Resistentes a outliers

Flexibilidade

Funcionam com dados ordinais

Menos poder estatístico que testes paramétricos, mas mais robustos

Comparação: Paramétrico vs Não Paramétrico

Teste t (Paramétrico)

✓ Dados normais

✓ Compara médias

✓ Maior poder estatístico

❌ Sensível a outliers

Mann-Whitney (NP)

✓ Dados não normais

✓ Compara medianas

✓ Robustos a outliers

❌ Menor poder estatístico
