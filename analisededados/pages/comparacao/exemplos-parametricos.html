Exemplos Práticos - Testes Paramétricos

Passo a passo para aplicar Teste t e ANOVA

Teste t de Student

Comparando eficácia de dois medicamentos

1

Definir Hipóteses

H₀: Não há diferença na eficácia entre os medicamentos
H₁: Há diferença na eficácia entre os medicamentos

2

Coletar Dados

Medicamento A: 85, 78, 92, 88, 76, 90, 84, 79, 87, 82
Medicamento B: 72, 68, 75, 80, 65, 78, 70, 74, 69, 77
Pontuação de eficácia (0-100)

3

Verificar Pressupostos

✓ Normalidade: Teste de Shapiro-Wilk (p > 0.05)

✓ Homogeneidade: Teste F (p > 0.05)

✓ Independência: Amostras independentes

4

Calcular Estatística

Fórmula: t = (Média_A - Média_B) / √(s²/n_A + s²/n_B)
Resultado: t(18) = 4.32, p = 0.0004

5

Interpretar Resultado

Conclusão: Rejeitar H₀ (p < 0.05)
Significado: Medicamento A é significativamente mais eficaz

Resumo do Exemplo

Medicamento A

Média: 84.1

Medicamento B

Média: 72.8

Diferença significativa: p = 0.0004

ANOVA

Comparando três métodos de ensino

1

Definir Hipóteses

H₀: Todos os métodos têm a mesma eficácia
H₁: Pelo menos um método é diferente

2

Coletar Dados

Método Tradicional: 65, 70, 68, 72, 75
Método Digital: 78, 82, 85, 80, 88
Método Híbrido: 90, 88, 92, 85, 95
Notas finais (0-100)

3

Verificar Pressupostos

✓ Normalidade: Teste de Shapiro-Wilk em cada grupo

✓ Homogeneidade: Teste de Levene (p > 0.05)

✓ Independência: Amostras independentes

4

Calcular ANOVA

Fórmula: F = Variância entre grupos / Variância dentro dos grupos
Resultado: F(2,12) = 15.67, p = 0.0005

5

Teste Post-Hoc

Tukey HSD:
• Tradicional vs Digital: p = 0.008
• Tradicional vs Híbrido: p = 0.0001
• Digital vs Híbrido: p = 0.045

6

Interpretar Resultado

Conclusão: Rejeitar H₀ (p < 0.05)
Significado: Métodos têm eficácias diferentes
Ranking: Híbrido > Digital > Tradicional

Resumo do Exemplo

Tradicional

Média: 70.0

Digital

Média: 82.6

Híbrido

Média: 90.0

Diferença significativa: p = 0.0005

Comparação: Teste t vs ANOVA

Teste t

✓ Compara 2 grupos

✓ Testa diferença entre médias

✓ Mais simples de calcular

❌ Limitado a 2 grupos

ANOVA

✓ Compara 3+ grupos

✓ Testa diferença geral primeiro

✓ Requer teste post-hoc

❌ Mais complexo

Quando Usar Cada Teste?

Teste t: Comparar 2 grupos (ex: tratamento vs controle)

ANOVA: Comparar 3+ grupos (ex: diferentes métodos)

Pressupostos dos Testes Paramétricos

Normalidade: Dados devem seguir distribuição normal

Homogeneidade: Variâncias devem ser semelhantes

Independência: Observações devem ser independentes
