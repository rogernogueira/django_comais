Homogeneidade de Variâncias

Também conhecida como Homocedasticidade: a dispersão dos dados deve ser similar entre os grupos.

1. O que é?

Para comparar médias de forma justa, a "largura" das distribuições (variância) deve ser parecida.

Variâncias Homogêneas (Ideal)

Ambas as curvas têm a mesma "largura".

Variâncias Heterogêneas (Problema)

Curvas com larguras muito diferentes.

2. Como Testar?

Testes para verificar a igualdade de variâncias.

Teste de Levene

Mais robusto e comum.

Funciona bem mesmo se os dados não forem perfeitamente normais.

Teste de Bartlett

Mais sensível.

Requer que os dados tenham distribuição normal rigorosa.

Interpretação (Levene)

p < 0.05

Variâncias Diferentes

(Heterocedasticidade)

p > 0.05

Variâncias Iguais

(Homocedasticidade)

Objetivo: Queremos p > 0.05 para usar testes paramétricos padrão (como Teste t).

E se falhar?

Se as variâncias forem diferentes, use o Teste t de Welch (não assume variâncias iguais) ou um teste não paramétrico.
