Representação Gráfica

O que é a Distribuição Normal?

A distribuição normal, também conhecida como distribuição gaussiana, é uma das distribuições mais importantes na estatística. Ela descreve muitos fenômenos naturais e é fundamental para testes estatísticos paramétricos.

Propriedades Principais

✓ Simetria: A curva é perfeitamente simétrica em torno da média

✓ Centralidade: Média = Mediana = Moda

✓ Forma de sino: A curva tem formato característico de sino

✓ Assintótica: A curva se aproxima, mas nunca toca o eixo X

Regra Empírica (68-95-99.7)

68%

±1 desvio padrão

95%

±2 desvios padrão

99.7%

±3 desvios padrão

Esta regra mostra a porcentagem de dados que caem dentro de cada intervalo de desvios padrão da média

Importância nos Testes Paramétricos

Os testes paramétricos assumem que os dados seguem uma distribuição normal. Quando esta suposição é violada, os resultados podem ser inválidos, tornando necessário o uso de testes não paramétricos.
