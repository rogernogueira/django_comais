Outliers e Robustez

Entenda o que são outliers e por que testes não paramétricos são robustos

Impacto dos Outliers

Observe como a média se move com outliers, mas a mediana permanece estável

O que são Outliers?

⚠️ Valores extremos que se afastam significativamente do padrão geral

⚠️ Podem ser erros de medição ou fenômenos reais

⚠️ Distorcem a média e outros parâmetros

⚠️ Afetam testes paramétricos que usam médias

Por que Outliers são um Problema?

Outliers podem distorcer completamente os resultados de testes estatísticos que dependem de médias e variâncias. Eles "puxam" a média para sua direção, tornando as conclusões estatísticas inválidas.

Exemplo Prático: Renda

Com Outliers

30k, 35k, 40k, 45k, 50k, 100k, 200k

Média: 71k

(distorcida)

Sem Outliers

30k, 35k, 40k, 45k, 50k

Média: 40k

(realista)

A mediana permanece 45k em ambos os casos - robusta a outliers!

Por que Testes NP são Robustos?

✓ Usam postos (ranks) em vez de valores absolutos

✓ Comparam medianas que são resistentes a outliers

✓ Transformação dos dados reduz influência de valores extremos

✓ Não assumem distribuição normal que outliers violam

Comparação: Média vs Mediana

Média

✓ Usa todos os valores

✓ Boa para dados normais

❌ Sensível a outliers

❌ Pode ser distorcida

Mediana

✓ Usa valor central

✓ Robusta a outliers

✓ Melhor para dados assimétricos

❌ Ignora informação dos extremos

Quando Usar Cada Medida?

Média: Dados normais, sem outliers

Mediana: Com outliers, dados assimétricos

Testes NP: Quando usar mediana

Resumo Prático

Se há outliers → Use mediana e testes não paramétricos

Se dados são normais → Use média e testes paramétricos

Sempre verifique outliers antes de escolher o teste
