Verificando a Normalidade

Como saber se seus dados seguem uma distribuição normal?

1. Métodos Visuais

A primeira etapa é sempre visualizar os dados.

Histograma

Deve ter formato de "sino" (simétrico).

Q-Q Plot (Quantil-Quantil)

Os pontos devem seguir a linha diagonal.

2. Testes Estatísticos

Testes formais para confirmar a normalidade.

Shapiro-Wilk

Recomendado para amostras pequenas (n < 50).

É o teste mais poderoso para normalidade.

Kolmogorov-Smirnov

Recomendado para amostras grandes (n > 50).

Geralmente usado com a correção de Lilliefors.

Como Interpretar o P-valor?

p < 0.05

NÃO é normal

(Rejeita H0)

p > 0.05

É normal

(Não rejeita H0)

H0 (Hipótese Nula): Os dados seguem uma distribuição normal.
