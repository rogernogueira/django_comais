ANOVA - Análise de Variância

Comparação de médias entre três ou mais grupos independentes

Comparação de Múltiplos Grupos

Estatística F

F = Variância Entre Grupos / Variância Dentro dos Grupos

Quanto maior o F, maior a diferença entre os grupos

Testes Post-Hoc

Tukey HSD: Comparações múltiplas

Bonferroni: Controle conservador

Scheffé: Mais flexível

O que é ANOVA?

ANOVA (Analysis of Variance) é um teste estatístico paramétrico usado para comparar as médias de três ou mais grupos independentes. Ela testa se pelo menos uma das médias é diferente das outras, analisando a variabilidade dos dados.

Quando Usar ANOVA?

✓ 3+ grupos: Comparar três ou mais grupos

✓ Dados normais: Distribuição normal em todos os grupos

✓ Homogeneidade: Variâncias semelhantes entre grupos

✓ Independência: Grupos independentes entre si

Tipos de ANOVA

One-Way ANOVA

Um fator independente

Ex: Efeito de 3 dietas diferentes no peso

Two-Way ANOVA

Dois fatores independentes

Ex: Efeito de dieta E exercício no peso

Interpretação do Resultado

p < 0.05

Pelo menos uma média difere

p ≥ 0.05

Médias não diferem

Se p < 0.05, use testes post-hoc para identificar quais grupos diferem

Exemplos Práticos

Agricultura: Comparar rendimento de 4 fertilizantes

Medicina: Eficácia de 3 tratamentos

Educação: Desempenho de 5 métodos de ensino

Alternativa Não Paramétrica

Kruskal-Wallis

Use quando os dados não são normais ou têm outliers

Compara medianas ao invés de médias
