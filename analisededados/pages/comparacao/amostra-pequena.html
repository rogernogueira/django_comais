Amostra Pequena (n < 30)

Com poucos dados, a incerteza reina e a normalidade é difícil de provar.

A Incerteza dos Pequenos Números

Sem Teorema do Limite Central

Com n < 30, não podemos garantir que a distribuição das médias seja normal.

Testes de Normalidade Falham

Testes como Shapiro-Wilk têm baixo poder com amostras pequenas. Eles podem dizer que os dados são normais simplesmente porque não têm evidência suficiente para dizer o contrário.

"Ausência de evidência não é evidência de ausência."

A Abordagem Segura

Assuma Não Paramétrico

Se n é muito pequeno (ex: n=5 ou n=10) e você não tem certeza absoluta sobre a população de origem, é mais seguro usar testes não paramétricos.

Exceção: Dados Biológicos Claros

Se você sabe, por literatura anterior, que a variável (ex: altura humana) é normal, pode usar paramétricos mesmo com n pequeno.
